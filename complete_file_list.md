## üöÄ Quick Start with New Structure

### 1. **Setup Project**
```bash
# Clone and setup
git clone https://github.com/your-org/whatsapp-ai-bot.git
cd whatsapp-ai-bot
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 2. **Configure Keywords (Optional)**
Edit `config/keywords.json` to add/modify keywords:
```json
{
  "greetings": ["hi", "hello", "namaste", "your-language-greeting"],
  "acknowledgments": ["thanks", "ok", "noted", "your-acknowledgment"]
}
```

### 3. **Setup Environment**
```bash
cp .env.example .env
# Edit .env with your credentials
```

### 4. **Create FAQ CSV**
Create `data/faq_database.csv`:
```csv
Question,Answer,Category,Interface
How to reset password?,Go to Settings > Reset,Technical,CMS
```

### 5. **Build FAISS Index**
```bash
python scripts/build_faiss_index.py build
```

### 6. **Run Bot**
```bash
python main.py
```

## üîß How to Use Configurable Keywords

### **Method 1: Use Singleton Instance**
```python
from src.utils.message_helpers import get_message_helpers

helpers = get_message_helpers()

if helpers.is_pure_greeting(message):
    response = "Hello! How can I help?"
```

### **Method 2: Use Convenience Functions**
```python
from src.utils.message_helpers import (
    is_pure_greeting,
    is_paypal_query,
    extract_question_from_message
)

if is_pure_greeting(message):
    # Handle greeting
    pass
```

### **Method 3: Runtime Keyword Addition**
```python
helpers = get_message_helpers()

# Add new keywords dynamically
helpers.add_keywords_to_category("greetings", ["hey there", "wassup"])
```

### **Method 4: Get Keywords for Category**
```python
helpers = get_message_helpers()

# Get all greeting keywords
greetings = helpers.get_keywords_for_category("greetings")
print(greetings)  # ['hi', 'hello', 'hey', ...]
```
